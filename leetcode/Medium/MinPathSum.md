### MinPathSum

URL : https://leetcode-cn.com/problems/minimum-path-sum/

给定一个包含非负整数的 *m* x *n* 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

**说明：**每次只能向下或者向右移动一步。

**示例:**

```reStructuredText
输入:
[
  [1,3,1],
  [1,5,1],
  [4,2,1]
]
输出: 7
解释: 因为路径 1→3→1→1→1 的总和最小。
```



### 题解

- 经典动态规划

思路过程

```reStructuredText
初始化行列的格子代价
1,4,5
2,0,0
6,0,0

逐步取迭代前进，取最小值
1,4,5      1,4,5      1,4,5
2,7,0  ->  2,7,0  ->  2,7,6
6,0,0      6,8,0      6,8,7
```
