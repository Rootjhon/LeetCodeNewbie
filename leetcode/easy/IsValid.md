### 有效括号

URL : https://leetcode-cn.com/problems/valid-parentheses/

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。

**示例 1:**

```reStructuredText
输入: "()"
输出: true
```

**示例 2:**

```reStructuredText
输入: "()[]{}"
输出: true
```

**示例 3:**

```reStructuredText
输入: "(]"
输出: false
```

**示例 4:**

```reStructuredText
输入: "([)]"
输出: false
```

**示例 5:**

```reStructuredText
输入: "{[]}"
输出: true
```



### 题解

- 这里的括号不是数学意义上的大小括号，只需要逻辑配对
